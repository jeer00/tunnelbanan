# Tunnelbanan

A React singleplayer subway builder prototype set on a Stockholm-inspired map.

```bash
npm install
npm run dev
```

## Current Gameplay

- Choose a line, construction method, and service frequency.
- Click districts to extend the active line from its current endpoint.
- Use express mode to upgrade existing stops.
- Build around water crossings, which are more expensive.
- Watch budget, daily riders, coverage, and mood respond to the network.
- Complete the scenario objectives in the left panel.

Progress is saved in `localStorage`. Use Reset to start over.
